# AI-Based Disease Prediction and Healthcare Assistant

## Project Overview

This project predicts diseases based on symptoms using Machine Learning.

A Random Forest Classifier was trained on symptom data and integrated into a healthcare chatbot that can:

- Predict diseases from symptoms
- Display disease descriptions
- Suggest precautions
- Assist users through a conversational interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Machine Learning Model

Algorithm Used:

- Random Forest Classifier

Evaluation:

- Accuracy Score
- Confusion Matrix
- Classification Report
- Feature Importance Analysis

## Project Workflow

1. Data Preprocessing
2. Label Encoding
3. Train-Test Split
4. Random Forest Training
5. Model Evaluation
6. Disease Prediction
7. Healthcare Chatbot Integration
- Web Interface using Streamlit
- Voice-based Interaction
- Real-time Healthcare Assistance
